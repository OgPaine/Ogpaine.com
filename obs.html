Useful Plug-ins and Scripts

Plugins

StreamFX-https://obsproject.com/forum/resources/streamfx-for-obs®-studio.578/

Adds a ton of features such as shader,sharpen,blur,color grading,denoising,upscaling and sdf effects. sdf effects is useful of an easy cam drop shadow

Move transition-https://obsproject.com/forum/threads/move-transition.117568/

Really freaking cool plugin that lets you move sources around your scenes without scene transitions. Kinda difficult to use at first

win-capture-audio-https://obsproject.com/forum/threads/win-capture-audio.147240/

Lets you capture audio sources separately. Desktop sounds are a thing of the past in streams

Scripts

Advanced Timer-https://obsproject.com/forum/resources/advanced-timer.637/

Easy to set up timer using a text source

obs-filter-hotkeys-https://obsproject.com/forum/threads/obs-filter-hotkeys.132411/

Script allows you to set up hotkeys to toggle filters within obs

Default scene on startup-https://obsproject.com/forum/threads/default-scene-on-startup.131742/

You can have obs open up to any scene by default
